## Getting Started

### Creating Your Account
1. Visit the Python Pathfinder website
2. Click "Sign Up Free" button
3. Choose your username and password
4. Select your preferred theme (Cute or Deadly)
5. Verify your email address

### First Login
- Complete the interactive tutorial
- Learn basic navigation and controls
- Complete your first coding challenge
- Earn your first achievement

### Dashboard Overview
Your dashboard is your command center:
- **Progress Tracker**: See your learning journey
- **Quick Actions**: Jump into games or lessons
- **Stats Panel**: Track your scores and achievements

## Game Modes

### Single Player Mode
Learn at your own pace through structured challenges.

#### Level Progression
1. **Level 1-3**: Python Basics
   - Variables and data types
   - Basic operations
   - Input/output

2. **Level 4-6**: Control Structures
   - If/else statements
   - Loops (for/while)
   - Functions

3. **Level 7-10**: Advanced Topics
   - Lists and dictionaries
   - File handling
   - Error handling

#### Challenge Types
- **Code Completion**: Fill in missing code
- **Bug Fixing**: Find and fix errors
- **Algorithm Design**: Solve problems with code
- **Project Building**: Create complete programs

### Multiplayer Mode
Compete against other players in real-time battles.

#### Battle Types
1. **1v1 Duel**: Head-to-head coding challenge
2. **Team Battle**: 2v2 or 3v3 team competitions
3. **Tournament**: Bracketed elimination matches
4. **Speed Run**: Complete challenges against the clock

#### Match Flow
1. **Lobby**: Join or create a room
2. **Challenge Selection**: Choose or get random challenge
3. **Coding Phase**: Write solution within time limit
4. **Evaluation**: Code is automatically tested
5. **Results**: Scores calculated and winner determined

#### Rating System
- **Elo Rating**: Starts at 1000
- **Win/Loss**: Affects rating based on opponent strength
- **Streaks**: Bonus points for consecutive wins
- **Divisions**: Bronze, Silver, Gold, Platinum, Diamond
